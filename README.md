# Progressive Web App and Mongodb Budget Tracker

## _[ Deployed PWA Budget Tracker app](https://offlinebudgettrack.herokuapp.com/)_

## Description

Budget Tracker application to allow for offline access and functionality.

The user will be able to add expenses and deposits to their budget with or without a connection. When entering transactions offline, they should populate the total when brought back online.

Offline Functionality:

- Enter deposits offline

- Enter expenses offline

When brought back online:

- Offline entries should be added to tracker.

## Installation

_To install this application, make sure you clone the repo to your device. Install node. Run npm init. Run npm install. Install MongoDb_

## npm install

- ## Usage

Just enter name of the transaction and amount than hit Add Funds or Subtract Funds, if you have a network connection it will add transaction into our mongo database, or if no network connection it will save entrees in indexdb, after connected back to network it will automatically push our entrees into our mongo database.
